# 🔢 Number Guessing Game

A sleek, modern, and fully responsive single-file web application built with vanilla HTML, CSS, and JavaScript. Challenge your intuition by finding the secret number between 1 and 100 in 10 attempts or less!

## ✨ Features
* **Proximity Hints:** The game doesn't just say High/Low—it intelligently scales your hints (e.g., "Too Low" vs a close "Low") based on how close you are to the number!
* **Glassmorphic Dark Theme:** A stunning, ultra-modern UI designed with custom CSS variables, smooth transitions, and responsive grid layouts.
* **Smart Input Validation:** Built-in safeguards preventing empty submissions, decimal numbers, or out-of-range inputs from wasting your turns.
* **Dynamic State Resets:** Instantly replay without refreshing the browser page using the interactive "Play Again" engine.
